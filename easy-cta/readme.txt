== Description ==

Use Easy CTA to create customized CTA blocks for your WordPress Website.

= Features =
* CTAs can be used on multiple pages but can be edited in one place
* Compatible with Gutenberg Blocks
* Allows for the use of background images or colors
* Easy for administrators to use, and easy for developers to customize
* Adding your own block-cta.php template to your_theme/template-parts/blocks will allow you to customize the block output.


== Installation ==

From your WordPress dashboard

**NOTE: This plugin requires ACF Pro**

1. **Visit** Plugins > Add New
2. **Search** for "Advanced Custom Fields PRO"
3. **Activate** Advanced Custom Fields from your Plugins page
4. **Search** for "Easy CTA"
5. **Activate** Easy CTA
6. **Move ACF JSON** Copy the easy-cta/acf-json files from the plugin, and add them to the acf-json folder in your theme.
